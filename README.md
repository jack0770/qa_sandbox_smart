# QA Sandbox Smart Contract

Solana smart contract на Anchor Framework - защищённый счётчик с тестами.

## Возможности

- ✅ Инициализация счётчика с названием
- ✅ Увеличение/уменьшение значения
- ✅ Установка конкретного значения (0-1,000,000)
- ✅ Защита от overflow/underflow
- ✅ Контроль доступа (только владелец)
- ✅ 8 тестов (функциональные + безопасность)

## Установка

```bash
# Установить зависимости
yarn install

# Собрать программу
anchor build

# Запустить тесты
anchor test
